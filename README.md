# 📝 AI 作业批改助手

> **本项目由阿里云ESA提供加速、计算和保护**

## 📖 项目简介

**AI 作业批改助手** 是一个基于阿里云 AI 模型的智能作业批改工具。通过 ESA 边缘函数代理调用阿里云通义千问等 AI 模型，实现作业的智能批改、评分和评语生成，帮助教师和学生快速获得作业反馈。

### 核心功能

- 📝 **文本作业批改**：支持直接输入文本内容进行批改
- 📄 **文件作业批改**：支持上传文档、图片等文件进行批改
- ⚡ **边缘加速**：利用 ESA 边缘节点，实现低延迟批改
- 📊 **智能评分**：自动评分（0-100分）和等级评定（优秀/良好/及格/不及格）
- 💬 **详细评语**：生成针对性的批改评语和建议

## 🛠️ 技术栈

- **前端框架**：Vue 3 + TypeScript + Vite
- **UI 组件库**：Element Plus
- **边缘计算**：阿里云 ESA 边缘函数
- **AI 模型**：阿里云通义千问（Qwen）
- **部署平台**：阿里云 ESA Pages

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 8.0.0

### 安装步骤

1. **克隆项目**
```bash
git clone <your-repo-url>
cd ai-homework-grader
```

2. **安装依赖**
```bash
npm install
```

3. **配置环境变量**

创建 `.env` 文件：
```env
# 边缘函数 URL（如果未配置，将使用模拟模式）
VITE_EDGE_FUNCTION_URL=/api/grader

# 开发模式：设置为 true 时始终使用模拟数据
VITE_USE_MOCK=false
```

**注意**：如果边缘函数未部署，系统会自动使用模拟模式，方便本地开发和测试。

4. **启动开发服务器**
```bash
npm run dev
```

5. **构建生产版本**
```bash
npm run build
```

## 📦 部署到阿里云 ESA Pages

### 部署步骤

1. **准备代码仓库**
   - 将代码推送到 GitHub 公开仓库

2. **部署边缘函数**
   - 在 ESA 控制台创建边缘函数
   - 将 `edge-function/grader.js` 部署到边缘节点
   - 配置环境变量：
     - `ALIYUN_ACCESS_KEY_ID`
     - `ALIYUN_ACCESS_KEY_SECRET`
   - 获取边缘函数的访问 URL

3. **在 ESA Pages 创建站点**
   - 登录 [阿里云 ESA 控制台](https://esa.console.aliyun.com/)
   - 进入「Pages」服务
   - 点击「新建站点」，选择「GitHub 仓库」
   - 连接你的 GitHub 账号并选择仓库

4. **配置构建设置**
   - **构建命令**：`npm install && npm run build`
   - **输出目录**：`dist`
   - **Node 版本**：选择 18.x 或更高

5. **配置环境变量**
   - 在 ESA Pages 站点设置中添加：
     - `VITE_EDGE_FUNCTION_URL`（边缘函数的访问 URL）

6. **部署**
   - 点击「部署」按钮
   - 等待构建完成
   - 获取部署后的访问 URL

## 🎯 使用说明

1. **配置 API Key**（可选）：
   - 点击右上角「配置 API Key」按钮
   - 输入阿里云 AccessKey ID 和 Secret
   - 配置保存在浏览器本地，不会上传到服务器

2. **提交作业**：
   - **文本输入**：切换到「文本输入」标签，输入作业内容
   - **文件上传**：切换到「文件上传」标签，上传作业文件（支持 .txt, .doc, .docx, .pdf, 图片等）

3. **开始批改**：
   - 点击「开始批改」按钮
   - 等待 AI 处理（约 1-2 秒）
   - 查看批改结果

4. **查看结果**：
   - **评分**：圆形分数显示（0-100分）
   - **等级**：优秀/良好/及格/不及格
   - **评语**：详细的批改评语和建议
